# AH_03_신정호 JavaScript 실습과제

## 구현 내용
1. 콘솔 계산기 프로그램
2. 최종 합격 판정 프로그램

## 파일 설명
- `AH_03_신정호_console_cal.js` : 콘솔 계산기 프로그램
- `AH_03_신정호_judgement_pass.js` : 최종 합격 판정 프로그램

## 실행 방법
각 JavaScript 파일을 브라우저 개발자도구 Console에서 실행하거나,
로컬 환경에서 실행하여 결과를 확인할 수 있습니다.

## 출력 내용
- `AH_03_신정호_console_cal.js`
  - 숫자 2개와 연산자(+,-,*,/)를 사용해서 계산 결과를 출력합니다.
  - 덧셈, 뺄셈, 곱셈, 나눗셈 결과를 각각 `console.log()`로 확인할 수 있습니다.
  - 예를 들어 n1=10, n2=20일 때 결과는 다음과 같이 출력됩니다.
    - 덧셈: 30
    - 뺄셈: -10
    - 곱셈: 200
    - 나눗셈: 0.5

- `AH_03_신정호_judgement_pass.js`
  - 점수(score), 출석(attendance), 과제 제출 여부(submit)를 기준으로 최종 합격 여부를 판정합니다.
  - 조건에 맞으면 `"결과: 합격입니다."`를 출력하고, 조건에 맞지 않으면 `"결과: 불합격입니다."`를 출력합니다.
  - 예를 들어:
    - score = 85, attendance = 79, submit = true → 결과: 불합격입니다.
    - score = 90, attendance = 79, submit = true → 결과: 합격입니다.
