[AH_03_신정호 JavaScript 실습과제 제출용 코드]

JavaScript 실습 과제를 정리한 파일입니다.

구현한 기능
1. 버튼 클릭 이벤트
2. 텍스트 변경
3. 실시간 입력
4. 마우스 오버 / 아웃
5. 스타일 변경
6. 점수 판별

추가 기능
- 콘솔 계산기

파일 설명
- AH_03_신정호_js실습과제.html : 화면 구성
- AH_03_신정호_js실습과제.js : 기능 구현
- AH_03_신정호_제출용.txt : 제출용 설명

실행 방법
AH_03_신정호_js실습과제.html 파일을 브라우저에서 열면 실행
AH_03_신정호_js실습과제.js 파일이 연결되어 기능이 작동

콘솔 계산기 결과는 개발자도구 Console에서 확인할 수 있습니다.
